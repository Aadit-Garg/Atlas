# Atlas Studio 🖥️

Atlas Studio is the official visual window into Atlas Runtime.

## Composed Workers
- `atlas.core.logger`
- `atlas.core.config`
- `atlas.ui.window`
- `atlas.ui.workspace`
- `atlas.ui.theme`

## Features
- **Dashboard:** Overview of runtime health and active metrics.
- **Workers:** Browse installed workers, roles, and status.
- **Managers:** Inspect Atlas Suite managers.
- **Models:** Inspect abstract contracts.
- **Packages:** Package management view.
- **Settings:** Customize themes and runtime connections.

## Running Studio
```bash
python suite/studio/main.py
```
Or open http://localhost:5000 in your browser.
